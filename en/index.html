Home Assistant Server

Home Assistant is an open-source platform that allows you to control and automate various smart devices and systems in your home.

So, I installed Home Assistant OS on the PC that was given to me by the region, configured remote access via my Freebox Pop using Free's DNS servers, and set up various modules. For example, my lights will sync with alarms on my smartphone on weekdays before 8 am.

The advantage of this platform is that all data and processing are done locally (it is possible to activate a cloud option), which does not compromise my security and privacy.
